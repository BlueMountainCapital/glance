Glance

Easily add new values.

Searching for and finding new values is easy.

Easily define new pages.

An easy-to-use API for defining new pages. 

glance.page("cpu","CPU")
      .find("*.cpu")
      .asPercent(1.0)

All in the browser.

There are no dependencies on any processes running on the server.

Open source, easily extensible.

Using open source components, glance was designed so that users can easily develop new components, and make more use of the components.

Bring metrics in.

Easily integrate graphite metrics. Just adding a graphite metric handler will allow users to start pulling graphite metrics into their dashboards.
